# Deep Agent v0.8.0

Release date: 2026-07-13

## Goal

Make short questions fast while allowing large text/code and difficult project
work to use deeper, bounded reasoning without a silent terminal or fragile
single-request timeout.

## Added

- Local adaptive classification: `simple`, `standard`, `large`, and `deep`.
- Automatic dependency-aware starter plan for large/deep tasks.
- DeepSeek thinking toggle and `high`/`max` effort selected per task.
- SSE streaming and reassembly of `reasoning_content`, content, and tool calls.
- Elapsed Thinking UI with mode, round, current step, tool status, and streamed
  DeepSeek reasoning.
- Same-key exponential retry for transient network/timeout/408/5xx failures.
- Deep strategy and Task Graph preserved across short resume prompts.

## Fixed and hardened

- Bounded daemon queue timeout and full process-group termination for timed-out
  shell commands.
- Planner ID collision repair and visible plan truncation warning.
- Worktree Git branch recognition, invalid symbol filtering, and default TSX
  semantic indexing.
- Memory dedupe candidate indexing instead of global all-pairs comparison.
- Secret pattern redaction and private JSONL log permissions.
- `.project-agent/memory` is ignored and inaccessible to project tools.
- Reject `.project-agent` and source-file symlink escapes.
- Reject Queue ID traversal and reload the canonical Queue after locking.
- Unique temporary files for Session/Queue writes and latest-by-mtime recovery.
- Per-session resume lease prevents concurrent turns from replaying one Session.
- Unique atomic Context/Workspace cache writes and locked first project setup.
- Exact daemon argv/starttime identity and queue process-group termination.
- Bounded shell output, HTTP body/headers, document input, browser downloads,
  and MCP tool pagination.
- Deny Docker root/socket/device/host-namespace escape variants in normal/YOLO
  policy.

## Verification

- 86 pytest cases, including real PTY interaction.
- Ruff lint and format checks.
- Python compileall.
- Streaming/tool-call assembly, retry, partial-stream no-replay, concurrency,
  path traversal, symlink, performance, and process cleanup regressions.

## Rollback

Checkout tag `v0.7.1` and reinstall the editable package. Configuration
additions are backward-compatible and can remain in place.
